## Repository Structure

Repositories follow a consistent naming convention:

```
<year>-<ctf-event>-<category>-<challenge>
```

| Prefix | Description |
|---|---|
| `writeup-` | Post-competition writeups — solutions and methodology explained |
| `challenge-` | Original challenges authored for CTF events |
| `tool-` | Custom tools and scripts developed for CTF purposes |
